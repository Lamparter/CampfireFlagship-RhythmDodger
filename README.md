# `CampfireFlagship-RhythmDodger`

*Simple rhythm dodger game for Campfire Flagship*

---

> Game design

| **Category** | **Choice** | **Notes** |
|--------------|------------|-----------|
| **Resolution** | `800 Ã— 450` | |
| **Beat BPM** | `120 BPM` | One beat every 0.5 seconds |
| **Beat Interval** | `60BPM` | Used for spawning obstacles + timing UI |
| **Player** (player model) | *Default on ground with jump* | Moves freely within bounds (+gravity effect) |
| **Obstacles** (enemy model) | *Rectangles moving left* | Spawn on every beat |
